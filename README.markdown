#4. Description
Proxy Server network scanner and tunnelling tool.
#5. Usage
-m Script mode ( tunnel | scan )  
-v Verbose Mode  

-p Proxy Host:Port  
-u Proxy Username  
-w Proxy Password  

-r Remote Host:Port  
-l Local Host:Port  
 
-s Scan mode ( connect | get )  
-t Address to scan  
-o Port Range  
-n Number of Threads  

-g The obligatory evilness
